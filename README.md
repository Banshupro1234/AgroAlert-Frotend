# Agro Alert Web Application

## Overview
The Agro Alert Web Application is designed to provide real-time weather alerts and crop protection insights for farmers and agricultural enthusiasts. The application allows users to receive notifications about weather conditions that may impact their crops, including frost warnings, rainfall alerts, and more.

## Project Structure
```
agro-alert-webapp
├── index.html          # Main HTML document for the web application
├── weather-dashboard.html  # Secondary page displaying weather information
├── css
│   └── styles.css     # Styles for the web application
├── js
│   ├── main.js        # Main JavaScript file for user interactions
│   └── weather-api.js  # JavaScript file for interacting with the weather API
├── images
│   └── download.jpg    # Image used in the web application
└── README.md           # Documentation for the project
```

## Features
- Real-time weather alerts based on user input (state or pincode).
- Dynamic content display on the weather dashboard.
- User-friendly interface with responsive design.
- Precautionary measures and seasonal tips for crop protection.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd agro-alert-webapp
   ```
3. Open `index.html` in a web browser to view the application.

## Usage
- Enter your state or pincode in the provided input fields to receive local weather alerts.
- Click the "Get State Alerts" or "Get Local Alerts" button to navigate to the weather dashboard.
